# NormAI

NormAI es una plataforma académica basada en Django para la gestión de la norma ISO 9001:2015,
integrando técnicas de Inteligencia Artificial para análisis, apoyo a decisiones y automatización
de procesos de calidad.

El proyecto se desarrolla dentro del marco del grupo IMAS Lab (Intelligent Models & Applied Systems).

## Stack tecnológico
- Backend: Django
- Base de datos: PostgreSQL
- IA: OpenAI API

## Estado del proyecto
Este repositorio contiene una base funcional desarrollada en TFG previos.
El objetivo actual es su extensión y consolidación académica.
